
# Restaurants Landing Page

Simple Angular 12 landing page with angular material


## How to use
1. Clone the repository with git clone
2. npm install
3. npm serve
4. open browser on http://localhost:4200/

That's it

## Frameworks
* Angular cli 12.2.18
